# Smart Productivity Dashboard
## Overview
Smart Productivity Dashboard is a responsive web application developed as part of ApexPlanet Full Stack Web Development Internship Task-2.
The project combines a Contact Form with Validation and a Dynamic To-Do List Application using HTML, CSS, and JavaScript.
## Features
### Contact Form
* Name Validation
* Email Validation
* Message Validation
* User-Friendly Alerts
### To-Do List
* Add New Tasks
* Delete Tasks
* Dynamic Task Counter
* Real-Time Updates
### User Interface
* Modern Responsive Design
* Mobile-Friendly Layout
* Interactive Components
* Clean and Professional Appearance
## Technologies Used
* HTML5
* CSS3
* JavaScript
